KEY FEATURES
================================================================================

✅ Marketplace Core
   • Publish food surplus lots
   • Search with multiple filters
   • Place bids on lots
   • Dynamic pricing (decreases as expiry approaches)

✅ Legal Compliance (Ley 1/2025)
   • 8-level use hierarchy (prevention → disposal)
   • Automatic validation by product state
   • Auto-generated legal documents
   • Full traceability

✅ Predictive Matching
   • AI recommends best buyers
   • Geographic proximity scoring
   • Product prediction

✅ Environmental Impact
   • CO2 calculations (LCA-based)
   • Impact equivalencies
   • Dashboard metrics

✅ Full Stack
   • Backend: FastAPI + SQLAlchemy + PostgreSQL + PostGIS
   • Frontend: Modern responsive SPA (HTML/CSS/JS)
   • Database: 8 tables + 4 views + sample data
   • DevOps: Docker + Docker Compose

API ENDPOINTS (18 TOTAL)
================================================================================

Generators:    POST /generadores, GET /generadores/{id}
Receivers:     POST /receptores, GET /receptores/{id}
Lots:          POST /lots, GET /lots, GET /lots/{id}
Bids:          POST /bids, GET /bids/{lot_id}
Transactions:  POST /transactions, GET /transactions/{id}
Compliance:    GET /compliance/{id}, GET /compliance-hierarchy
Matching:      GET /matches, GET /price-suggestion
Metrics:       GET /stats, GET /carbon-footprints, GET /health

Full documentation: http://localhost:8000/docs (after running)
